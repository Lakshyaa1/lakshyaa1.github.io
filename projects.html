Wearables BLE Signal Processing

ParkinSense

Built during my IIT Bombay internship, ParkinSense is a wearable neurological monitoring platform for Parkinson's disease using IMU and PPG sensing on the XIAO nRF52840 Sense.

Worked on embedded firmware, BLE streaming, and real-time processing pipelines for tremor monitoring, heart-rate and HRV estimation, SpO2 estimation, and activity tracking.

The system also supports live dashboards, dataset generation, and offline analysis, making it a strong base for research-oriented digital biomarker development.

UWB Zephyr RTOS Embedded Systems

UWB-Based Ranging & Localization Firmware

Built custom Zephyr RTOS firmware for DW3000 / DWM3001C nodes to develop an open ultra-wideband localization stack beyond vendor firmware limitations.

Completed 2D and 3D DS-TWR ranging and localization, and developed TDoA firmware with UWB-based clock synchronization between anchors.

Achieved about 20 cm inter-anchor synchronization accuracy for two anchors, but scaling beyond that produced unstable values, so the project concluded with DS-TWR fully demonstrated and TDoA partially validated in practice.
